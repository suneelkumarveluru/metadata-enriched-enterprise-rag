# Contributing

Thank you for your interest in contributing to this project.

This project explores metadata-enriched and policy-aware Retrieval-Augmented Generation (RAG) for enterprise data platforms.

## Project Focus

The project focuses on:

- Enterprise RAG
- Metadata-enriched retrieval
- Policy-aware access filtering
- Data governance
- Audit logging
- RAG evaluation
- AI-ready data platforms

## Ways to Contribute

You can contribute in the following areas:

- Improve documentation
- Add test cases
- Improve metadata schema
- Add vector database support
- Add Chroma or FAISS integration
- Improve policy filtering logic
- Add RAG evaluation metrics
- Add Streamlit demo UI
- Add architecture diagrams
- Add research references

Create a virtual environment:

python -m venv .venv

Activate the virtual environment:

# Windows
.venv\Scripts\activate

# Mac/Linux
source .venv/bin/activate

Install dependencies:

pip install -r requirements.txt

Run the demo:

Contribution Guidelines

Before making a contribution:

Keep the change focused and small.
Use clear file names and commit messages.
Add or update tests when changing logic.
Update documentation when adding a new feature.
Avoid using confidential or client-specific data.
Use only synthetic or public data.
Commit Message Examples
docs: improve metadata schema documentation
feat: add vector retrieval module
test: add policy filter test cases
chore: update dependencies
Research Integrity

This project is intended for practical research and educational use.

Do not include:

Confidential client data
Private company documents
Personal health information
Financial account data
Proprietary source code
Credentials or secrets
